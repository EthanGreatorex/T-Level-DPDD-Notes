Calculate Trends for a Lunch Item

  • A step by step representation of a function
  • item, start_date & end_date are arguments passed into this function
# Simply read in our data file

df = pd.read_csv("Task4a_data.csv")

# This will reduce our data frame to a smaller one where only the rows that have our item and lunch as a service are selected

lunch_df = df.loc[(df['Menu Item'] == item) & (df['Service'] == "Lunch")]

# This will filter the data frame and remove any columns that aren't within start_date and end_date.

lunch_range_df = lunch_df.loc[:,start_date:end_date]

# The reult will look something like this:
01/01/2021  02/02/2022  03/03/2023 ... 27/12/2027  28/12/2028

   45          56           99     ...    157          12

# As you can see, it has removed the 'Menu Item' & 'Service' columns as they come before the dates

# This will transform the data frame into two columns

 lunch_range_df = lunch_range_df.iloc[0]

# The result will look something like this:
 01/01/2021  45

 02/02/2023  56

 03/03/2023  99

 ...

 27/12/2027  157

 28/12/2028  12

# Now the dates are the index of our data frame, we can plot it :)

lunch_range_df.plot(kind='line',marker='o',color='purple',figsize=(12,6))


# Make sure we include details!
plt.title(f"Trends Between {start_date} & {end_date} for {item} at Lunch")

plt.xlabel("Date")

plt.xticks(rotation=15)

plt.ylabel("Amount of Sales")

plt.tight_layout()

plt.show()

Calculate Highest Total & Highest Average

  • A step by step representation of a function
  • start_date & end_date are passed into the function as arguments
# Simply read in our data file
df = pd.read_csv("Task4a_data.csv")

# Create two data frames. 
# The first one will calculate the sum
# The second will calculate the mean (average)
# numeric_only stops pandas from trying to add strings & numbers together

total_df = df.groupby('Menu Item').sum(numeric_only=True)
average_df = df.groupby('Menu Item').sum(numeric_only=True)

# Example output from total_df:
Menu Item   01/01/2021  02/02/2022
Nachos         50          15
Burgers        77          999

# Now we need to filter down the dataframe to our dates
# During the same line of code, we sum together each row (axis=1)

total_df = total_df.loc[:,start_date:end_date].sum(axis=1, numeric_only=True)
average_df = average_df.loc[:,start_date:end_date].sum(axis=1, numeric_only=True)

# Example output from total_df:
Menu Item
Brisket    419
Burger     491

# As you can see, all dates & values have be added up for each menu item

# The menu item is our index, this means we can plot a bar chart :)
# I will only plot total_df for the example

total_df.plot(kind='bar',color='purple',figsize=(10,5))

# Add details!
plt.title(f"Total Sales Between {start_date} & {end_date}")

plt.xlabel("Menu Item")

plt.xticks(rotation=15)

plt.ylabel("Total Sales")

plt.grid()

plt.tight_layout()

plt.show()
