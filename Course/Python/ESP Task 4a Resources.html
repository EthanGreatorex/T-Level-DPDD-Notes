Pandas Methods & Descriptions

Method Description
.index returns the index of the data frame
.to_string returns the string version of the data frame
.loc[] returns one or more rows if the condition is met
.head(n) shows the first n rows of the data frame
.tail(n) shows the last n rows of the data frame
describe() shows the summary statistics of the data frame
pd.to_datetime() converts the column passed into a datetime object
df['column'].isin(['Monday','Friday']) returns a new data frame if the column row is in the list
.idxmax() identifies the row index with the highest value
.sum(numerical_only=True) sums the rows together, makes sure strings aren't added to integers
.groupby('column') groups the data frame by that column. The column becomes the index

Scenario One

  • You are provided a data frame that contains airports & the amount of passengers from that airport on each day
  • You must plot the trends for a specific airport 'DUB'

Preview of the Data Frame

Coded Solution

import pandas as pd

from matplotlib import pyplot as plt
********
plt.style.use('dark_background')

df = pd.read_csv('Task4a_data.csv')

# Get the trends over time for dublin airport

dublin_df = df.loc[df['From'] == 'DUB']

# Get the dates from the columns

dates_columns = df.columns[3:]

dublin_trends = dublin_df.groupby('From')[dates_columns].sum()

plt.figure(figsize=(14,7))

# Plot the trends on a chart

plt.plot(dates_columns,dublin_trends.loc['DUB'],color='purple')

plt.title('Trends over time for Dublin Airport')

plt.xlabel('Date')

plt.ylabel('Passenger Numbers')

plt.xticks(rotation=90)

plt.tight_layout()

plt.show()

Scenario Two

  • Check for any negative numbers inside the data frame

Coded Solution

try:
	df = pd.read_csv("Task4a_data.csv")
except FileNotFoundError:
	print("Error whilst reading in data file...")
	exit()
else:
	print("\nChecking data for negative numbers...")
	if (df[['Tickets','Gift Shop','Snack Stand','Pictures']] < 0).any().any():
		print("Data set contains negative values...")
		print("Exiting...")
		exit()
	else:
		print("Data set has been verified... no negative numbers present.\n")
  • I have specified the columns which contain numerical numbers
    • If no columns are specified, there will be an error if a string column is found
